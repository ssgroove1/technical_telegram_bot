# Тех. бот поддержка в телеграмм
Технический телеграмм бот проект для интернет-магазина "Продаем все на свете".

## О проекте
- Проект представляет собой бота технической поддержки для телеграмма, который отвечает на часто задаваемые вопросы пользователей, предоставляет решения проблем и направляет запросы к специалистам при необходимости.

- Есть 2 отдела - программисты, которые могут починить что-то, если сломался сайт или оплата, и отдел продаж, который разбирается со всеми проблемами с товарами.

## Требования к проекту
Выполнено - ✔️ ; В процессе - ⏳ ;
Не выполнено - ❌
<ol>
  <li>⏳ Автоматический ответ на часто задаваемые вопросы пользователей. Вопросы и ответы на них в этом файле.</li>
  <li>⏳ Интеграция с базой данных - нужно хранить все запросы к специалистам в отдельной таблице.</li>
  <li>⏳ Возможность обработки текстовых сообщений (будет здорово, если и голосовых сообщений тоже, но необязательно).</li>
  <li>⏳ Интуитивно понятный и привлекательный интерфейс для пользователей.</li>
  <li>⏳ Есть документация по использованию бота для администраторов.</li>
</ol>
